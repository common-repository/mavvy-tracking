Mavvy Tracking is a plugin that has the possibility of tracking visits and sales using technologies behind Mavvy LLC

== Description ==

Mavvy Tracking plugin is an easy way to integrate the mavvy tracking platform in your Wordpress install. This plugin will handle
integrating the required embed code on all pages, and reporting sales data to the mavvy platform.

Data can be viewed in the mavvy dashboard available [here](https://app.mavvy.co/).

== Installation ==

= Minimum Requirements =

* WordPress 4.1 or greater
* PHP version 5.2.4 or greater

= Automatic installation =

Automatic installation lets Wordpress handle the file transfers and can be done without leaving your web browser.
To do an automatic install of Mavvy Tracking, log in to your WordPress admin area and navigate to the Plugins menu. Click Add New, and in the
search filed type "Mavvy Tracking" and click Search Plugins.

Once you've found our plugin, you can install it just by clicking "Install Now".

= Manual installation =

The manual installation can be done by downloading our plugin and uploading it through a FTP application to your Wordpress hosting. You
can find instructions on how to do that in the [WordPress codex](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

From time to time, we release updates that fix issues and implement new features. Your Wordpress install will automatically notice you about available
updates so you can install them. Please make sure that you backup your site before updating, just in case.
